# Project Documentation

## Objective
Build a Grade Management System using Java 8 with OOP principles.

## Data Structures Used
- Array (for marks)
- ArrayList (for storing students)

## Algorithms
- Average = Sum of marks / Number of subjects
- Grade classification using if-else ladder

## OOP Concepts
- Encapsulation
- Modular design
- Separation of concerns

## Future Enhancements
- File persistence
- Database integration
- GUI version
