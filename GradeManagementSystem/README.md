# Grade Management System

## Overview
A Java 8 compatible console-based application to manage student grades, calculate averages, assign grades, and generate reports.

## Features
- Add student marks
- Calculate average, highest, lowest
- Grade categorization (A+, A, B, C, D, F)
- Search student
- Performance report
- Input validation
- ArrayList-based storage

## How to Run

1. Navigate to src folder
2. Compile:
   javac *.java
3. Run:
   java GradeManagementSystem
